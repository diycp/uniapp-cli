# 一套用于 SPA 开发 和 Native APP 开发解决方案

### 版本更新提示

还没有任何版本发布

---

### 安装

```
npm install qmox -g
```
---


### 使用

在本地终端进入项目位置 然后使用 ```$ qmox init``` 命令 选择需要应用的框架程序, 
更多的的命令使用帮助可以使用 ```$ qmox --help``` 命令 获取更多帮助信息

  update [up] - Update qmox framework modules
